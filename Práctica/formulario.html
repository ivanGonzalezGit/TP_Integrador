Los formularios se usan para obtener información por el ingreso de datos del usurario. Contienen elementos anidados como etiquetas y campos.













  • El atributo action de la etiqueta form enlaza el formulario con otra url
  • El método de envío get muestra los datos enviados en la url, no hay q usarlo si se trabaja con contraseñas.
  • El método post no muestra los datos al enviarlos
